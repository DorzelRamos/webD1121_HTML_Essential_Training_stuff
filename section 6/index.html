Audio

The audio element is used to put audio in the website.

If you add the control attribute within the audio element it uses the browser to make a audio bar.

It's similar to the img element as it also uses src attribute

Here are some other attributes:

  • Loop

    • Loops audio after it finishes playing

  • Autoplay

    • Autoplays the audio immediately after the page loads

Video

You can use the video element to put videos within the websiet. It requires a src and a controls attribute.

There exists thigns called codecs that can be used to play videos, but most are patented. It allows you to source your videos in different ways to different devices across the internet.

Captions and subtitles

When audio is not enough, you can use a track element inside a video element to create subtitles for the video while it's playing.
