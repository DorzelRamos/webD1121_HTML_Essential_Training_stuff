The HTML page

The HTML File

The first file thats returned in response to a request for a webpage

When starting the HTML page there's several key items you need for it.

doctype decleration

The doctype decleration is what declares which era the HTML file is from.

The head element

The head element contains info the browser needs to know, but doesn't display it on the page

The body element

The body element containst he info and the content that will be displayed on the page

What goes in the document head?

The meta element only gose inside the head element. It conveys the meta data of the page.

Then theres the title which gives the browser the webpages name.

The link element is used to link to a range of other assets that we want to load.

The rel attribute tells the browser whihc kind of asset it is.

The href attribute provides the URL to the asset.

The script element is what's used to tell the browser to load in the JavaScript file. It doesn't have to go inside the head, it can be at the end of the HTML after everything is loaded.
