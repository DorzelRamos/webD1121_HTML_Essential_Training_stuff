Links

To write a link you use an anchor element, or a. It's written with an href, then you put an ="site page" and then add a link at the end.


The href stands for Hypertext reference. You can also wrap text around links. You can even put other URLs inside the anchor element to link to other websites. HTTP stands for Hypertext protocol. HTTPS stands for Hyperlink Secure.

URL Paths

Whenever we use a link on the same domain and same website, you can use whats called a relative URL to locate the files on the website. An absolute URL is where all the files are in the domain.


To make a URL from scratch you have to understand how files are organized. Certain techniques to organize files only work in HTML.
