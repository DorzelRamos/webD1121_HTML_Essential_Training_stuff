When to use tables?

You use the table when you're making a table. But a long time ago there wasn't CSS to create and stylize tables. Tabular data can be used for research data and organizing it. It contains the relationship of cells of data.

Here's what you need to create a table in HTML:

  • table

    • Wraps around the entire markup of table elements

  • tr

    • Also known as Table Row, it wraps around a set of elements, defining them as belonging to the same row. It's attributes are colspan, rowspan, and headers

  • th

    • Stands for Table Header. It defines the header for the column. It's attributes are the same as tr, except instead of headers its scope.

  • td

    • The Table Data is used to mark the actual bits of data.
