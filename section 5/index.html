Images

When you want to put an image in a website you use a IMG element

There are four attributes important to make an image:

  • The source attribute src
  • The alt attribute alt
  • The width attribute
  • The height attribute

SRC attribute

Tells the browser which file to load for the image to show up

ALT attribute

Acts as the placeholder whenever the image can't be seen. You can write about the image here

width attribute

Displays the images width. Can be in px, cm, inch, etc. Doesn't matter which goes first width or height

height attribute

Displayst he images height. Can be in px, cm, inch, etc. Doesn't matter which goes first height or width

The image formats

To put an image on a webpage it has to be a file the browser will understand. The goal is to have the highest quality photo in a small file size.

There are four main files that are commonly used:

GIF

  • Really good for compressing large colors
  • Has limited space of 256 colors
  • Can do transparency
  • Can have multiple frames and make short movies

SVG (Scalable vector graphic)

  • Good for logos, icons, etc.
  • Good for vector files

JPG (for compressing photos

  • Can be compressed
  • Size vs Quality

PNG (more new than GIF and JPG)

  • For images that need transparency
  • Good at compression
  • Good for photos and images
  • Depends when to use

Responsive Images

You can use HTML to send the same photo to different screens in different sizes. This works by leting the browser decide which file size is right for the device by giving it options. There's 1x, 2x, 3x, and so on and so forth.


You could also tell the browser to pick the image photos by device density and viewport width.

The size attributes for image elements

This lists which size image to use at which media querty

The srcset attributes for image elements

Allows you to specify mutiple files to be used based on either viewport width or image px width

The picture element

If you wrap an img element with a picture element you can decide which source elements to use for it.
