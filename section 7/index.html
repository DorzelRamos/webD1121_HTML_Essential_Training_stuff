Language attributes

You can change the language of your website within HTML.

For the lang attributes theres:

  • Language
  • Script
  • Alphabet

The lang attribute is universal so you can put it on any element.

dir element

A universal element which specifies where text should be written

The charset attribute

Defines the alphabet or set of characters for the script language

Unicode

A universal encoding standard for characters encompassing many languages

IMPORTANT!

Don't use div and spans for everything.

div

div is a block level element

span

span is an inline element

These two elements don't do anything until you point a CSS or JavaScript at them
